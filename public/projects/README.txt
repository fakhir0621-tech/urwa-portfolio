=== HOW TO ADD PROJECT SCREENSHOTS ===

1. Place your project screenshot images in this folder.
   Example filenames: screenshot1.jpg, screenshot2.jpg, screenshot3.jpg

2. Open src/App.js in VS Code

3. Find the "projects" array near the top (around line 90)

4. Find the "screenshots" field and add your image paths like this:
   screenshots: [
     '/projects/screenshot1.jpg',
     '/projects/screenshot2.jpg',
     '/projects/screenshot3.jpg',
   ],

5. Save the file — the gallery will update automatically!
